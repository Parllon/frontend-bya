# Logos

Marcas estaticas da Bya Marcondes.

- `logo.svg`    — monograma completo (BM + arcos), versao estatica do componente `Logo`.
- `favicon.svg` — icone usado na aba do navegador (a versao ativa fica em `public/favicon.svg`).

O componente React `src/components/ui/Logo/Logo.jsx` continua sendo a versao
"viva" (segue as cores dos tokens via CSS vars). Estes arquivos sao para usos
estaticos: e-mails, assinaturas, impressao, redes sociais, etc.
